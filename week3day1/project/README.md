# Phase 1, Week 1, Day 5 Friday Readme

Completed Build a Website as project. It is a IT company website providing details about there company products and services. Also includes details about the team and company location and a contact form.

Added following pages - 
- Home Page - index.html - Contains Hero of Website.
- About Page - about.html - Contains Detail about the Company.
- Services Page - services.html - Contains Details about Services Company provides.
- Team Page - team.html - Contains a details about the team behind the company.
- Contact Page - contact.html - Contains Form to Contact and Address Details as well as GMap Iframe.
